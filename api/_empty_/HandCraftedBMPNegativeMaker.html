def produceNegative(fileName: String, processType: String): Boolean

Handcrafted negative BMP producer for non compressed 24 bit BMP files. This does this in about 4 sets. step1: Open file input stream, returns Array of Bytes. step2: Parse header, check if this is a BMP file that meets conditions/assumptions to make BMP. step3: Make the pixels associated with pixel color negative. step4: Write to file. step5: If above was success return true otherwise throw error and return false.
